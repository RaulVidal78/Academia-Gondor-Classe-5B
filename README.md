# 🏰 Academia Gondor 5B

**Un anell per poder apendre tots junts**

Una aplicación web completa para gestionar los puntos de 21 alumnos (10 chicos y 11 chicas) con temática del Señor de los Anillos.

## ✨ Características

- 🎴 **21 Tarjetas de Alumnos** - Volteo 3D interactivo
- 🎵 **Sonidos Personalizados** - Monedas girando + trueno profundo
- 📊 **7 Pestañas Funcionales**:
  - Dashboard con tarjetas volteable
  - Reclutamiento de alumnos
  - Ranking en tiempo real
  - Galería de fotos
  - Estadísticas con gráficos
  - Personalización de colores y textos
  - Ajustamientos (volumen, brillo, tipografía)

- 💾 **Persistencia de Datos** - MySQL + LocalStorage
- 🌓 **Tema Claro/Oscuro**
- 📱 **Totalmente Responsive** - Móvil, tablet y desktop
- 🎨 **Temática Gondor** - Letras doradas, fondo Ciudad Blanca
